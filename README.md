# 项目简介

## 核心功能
- 基于YOLOv8的芯片表面缺陷检测，包含7个类别：颗粒、PIQ颗粒、划痕、涂层不良、PO污染、块状蚀刻、SEZ烧毁
- 支持4531张芯片表面缺陷数据集的完整训练
- 提供完整源码、标注数据集、训练好的模型及训练结果
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 实时显示目标位置、目标总数、置信度等信息
- 支持自定义置信度和IOU阈值
- 支持选择显示标签和原图
- 支持选择不同检测模型
- 支持批量检测并查看所有结果
- 支持检测结果保存

## 适用场景
- 芯片制造过程中的质量检测
- 电子元器件表面缺陷自动化识别
- 工业生产线上的视觉检测系统
- 研究机构对芯片缺陷的分析研究
- 需要高精度表面缺陷检测的工业场景

## 技术特点
- 使用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 依赖Python 3.9、opencv-python、PyQt5、ultralytics、torch1.9等技术栈
- 提供完整UI界面及图标资源
- 包含详细的项目运行教程
- 环境配置成功后可直接使用
